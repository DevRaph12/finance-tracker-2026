To keep the CSS organized, I used the following naming convention:

### Main Layout
- `#finance-card`: Main "floating" style container.
- `#resume-container`: Flexbox container for the summary cards.

### Components
- `.resume-item`: Individual summary cards (Incomes, Expenses, Balance).
- `.input-group`: Wrapper div for Label + Input within the form.
- `.translatable`: Trigger class for the translation engine.

### Colors (CSS Variables)
- `--bg-color`: Body background color.
- `--card-bg`: Card background color.
- `--text-color`: Primary text color.
